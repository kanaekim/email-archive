Final reminder: Messenger 14.1.0 availability

Dear Messenger Customer,

 

This is a final reminder that Aurea Messenger 14.1.0 is available for both MQ and ESB components.

This release helps customers stay aligned with current Java and security standards while improving overall platform stability.

Key benefits to keep in mind

  • ☕ Java 17 long-term support, helping reduce future upgrade pressure
  • 🔒 Security updates that support audit and compliance requirements
  • 🚀 Ongoing stability and compatibility improvements across the platform

Messenger 14.1.0 remains backward-compatible with components as far back as version 10.0, enabling modernization of the core platform without immediate changes to existing integrations.
