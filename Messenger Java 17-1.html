Messenger 14.1.0 is here:

Java 17 support, security updates, and improved stability

Dear Messenger Customer,

 

We’re excited to announce a major new release of Aurea Messenger (v14.1.0) across both MQ and ESB components.

 

This release focuses on modernizing the platform, strengthening security, and improving overall stability to support long-term operation in enterprise environments.

What’s new in Messenger 14.1.0

  • ☕ Comprehensive Java 17 support for both runtime and development environments
  • 🛠️ Java 17–ready Workbench, including updated tooling and improved error handling
  • 🔒 Security library updates addressing multiple CVEs
  • 🚀 Stability and compatibility improvements across the platform

What this means for you

Messenger 14.1.0 is built on Java 17, the current long-term support (LTS) standard. The release is backward-compatible with components as far back as version 10.0, allowing you to upgrade the core platform while continuing to support existing integrations.

If you’re running older components and want help planning the transition, our Professional Services team can assist with implementation and validation.
