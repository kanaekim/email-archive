Messenger 14.1.0 upgrade planning

Dear Messenger Customer,

 

Following up again regarding Aurea Messenger 14.1.0, now available across both MQ and ESB components.

This release delivers Java 17 support, security updates, and platform improvements intended to support long-term stability and maintainability.

 

Why this release is important

  • ☕ Java 17 LTS support, aligning Messenger with a modern, supported Java release
  • 🔒 Security library updates addressing multiple CVEs
  • 🚀 Runtime and Workbench improvements for greater stability and compatibility

Messenger 14.1.0 is backward-compatible with components as far back as version 10.0, allowing teams to upgrade the core platform while continuing to support existing integrations.
