Pivotal CRM 6.6.5.10 Now Available — New Outlook Add-In Features & Platform Improvements

Dear Pivotal Customer,

 

We are pleased to announce the release of Pivotal CRM 6.6.5.10. This release includes platform improvements, bug fixes, and new features. We recommend all customers apply this update at their earliest convenience.

 

PLATFORM IMPROVEMENTS

This release includes important updates to the Packaged Client that strengthen overall application security and stability. We recommend all deployments update promptly.

 

BUG FIXES

  • CheckBox controls no longer toggle when clicking in the surrounding area — only direct clicks on the checkbox square register.
  • Layout reflow restored — controls now correctly reposition when hidden dynamically in UX forms, matching behavior from versions 6.6.0 and 6.6.4.11.
  • TypeError on save resolved — concurrent modification warnings now display correctly when two users edit the same record in UX 6.6.5.7+.
  • SQL Server version detection fixed — SyncStream installer now properly recognizes SQL Server 2017, 2019, and 2022.

NEW FEATURES

  • Compose Mode Email Linking — Link emails to CRM records before sending, directly from the compose window in both Outlook Web and Outlook Classic.
  • Offline Mode — Continue linking emails and managing CRM data when disconnected from Pivotal CRM. Queued operations sync automatically when connectivity is restored.
