Upgrade Now: New List Manager Release Is Live

Dear List Manager Customers,

 

Just a friendly reminder that the latest version of List Manager is now available!

This release includes meaningful improvements in security, deployment simplicity, and overall platform support — and we don't want you to miss out.

If you haven't already, we encourage you to upgrade at your earliest convenience.

 

Native TLS 1.3. Stronger security. Less complexity.

List Manager now supports TLS 1.3 natively and is designed to achieve an A+ overall SSL/TLS security rating in standard deployments. That means you no longer need to rely on an NGINX reverse proxy solely to enforce modern transport security.

 

With fewer moving parts, your team benefits from stronger encryption, simpler architecture, and easier maintenance.

 

Built for current Microsoft platforms.
The latest release is fully compatible with:

  • SQL Server 2022
  • Windows Server 2022

This gives your team a clearer path to modernize infrastructure without waiting for core systems to catch up.

 

What this means for your team:
Cleaner deployments. Less workaround infrastructure. A stronger security posture without added operational burden.

 

Whether your priority is improving security, simplifying administration, or modernizing your environment, this release helps you move forward with confidence.
