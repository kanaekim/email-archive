How to get the best results with the Metabase Assistant

Hi everyone,

 

As a reminder, AdvocateHub Analytics is now in Metabase, and you can use the Gemini/GPT Assistant to build accurate reports quickly—no SQL required.

 

To build reports in Metabase, use the Gemini/GPT Assistant in the Question Builder (Notebook Editor). It will:

  • Select the right starting Model (e.g., Contacts, Events, Redemptions)
  • Provide step-by-step guidance for filters, joins, and summaries
  • Apply standard accuracy rules (exclude admins/locked accounts where relevant; use distinct counts on ID fields to avoid double-counting)
  • Recommend the best visualization (trend lines, leaderboards, tables)

For best results, include in your prompt:

  • Metric + timeframe + breakdown (e.g., “weekly engaged advocates, last 90 days, by campaign”).
